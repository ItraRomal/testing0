Данная программа должна делать следующие вещи:
1)Принимать при запуске два аргумента - номер первой страницы и номер последней
2)Вывести в консоль текущую информацию по проверке ссылок
3)По завершении работы оповестить о проделанной работе предложить создать репорт
4.1)При согласии пользователя программа создает в той же папке файл формата "reportДАТА.txt", 
содержащий список неработающих ссылок, разбитый на виды: отсутствие ссылки на отображаемой странице, 
отсутствие текста ссылки на страницы, отсутствие ссылки в файле страницы
4.2)В случае отказа та же информация будет выведена в консоли
